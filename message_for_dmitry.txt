Краткое описание проекта:

SMI#1 управляет 7 региональными отделами новостных каналов по всей России (около 70 каналов суммарно, некоторые с аудиторией 100-200к+ подписчиков). Я разработал для них бота, который автоматизирует процесс сбора и распределения новостного контента по тематическим топикам.

Технически бот представляет собой асинхронную систему на Python, которая:
• Мониторит в реальном времени десятки Telegram-каналов через Telethon API
• Автоматически пересылает все посты из отслеживаемых каналов в целевую группу
• Распределяет контент по тематическим топикам (темам) в группе согласно региональной принадлежности канала
• Генерирует дайджесты топ-новостей за выбранный период (неделя, месяц) с анализом популярности контента
• Имеет полноценный Telegram-интерфейс для управления (добавление/удаление каналов, создание регионов, статистика)
• Использует SQLite для хранения данных и статистики, кэширование для оптимизации производительности

Сейчас система активно используется отделом Дальнего Востока (Сахалин, Камчатка, Владивосток и другие регионы) — обрабатывает и пересылает около 1200+ сообщений в день из всех мониторируемых каналов.

Основные технологии: Python 3.10+, asyncio, Telethon, SQLite, YAML-конфигурация.
