Краткое описание проекта:

SMI#1 управляет 7 региональными отделами новостных каналов по всей России (около 70 каналов суммарно, некоторые с аудиторией 100-200к+ подписчиков). Я разработал для них бота, который автоматизирует процесс отбора и распределения новостного контента.

Технически бот представляет собой асинхронную систему на Python, которая:
• Мониторит в реальном времени десятки Telegram-каналов через Telethon API
• Использует систему приоритетов для классификации новостей (критические события, важные новости, общий контент)
• Автоматически распределяет контент по тематическим топикам в целевой группе
• Имеет полноценный Telegram-интерфейс для управления (добавление/удаление каналов, создание регионов, статистика)
• Использует SQLite для хранения данных и статистики, кэширование для оптимизации производительности

Сейчас система активно используется отделом Дальнего Востока (Сахалин, Камчатка, Владивосток и другие регионы) — обрабатывает около 1200+ сообщений в день, отбирая 7-10% наиболее релевантного контента.

Основные технологии: Python 3.10+, asyncio, Telethon, SQLite, YAML-конфигурация, Docker-ready архитектура.
